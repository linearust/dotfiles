Ultrathink.

Verify logic correctness.

## Goal

Code behavior matches intent — correct results, edge cases handled.

## Checks

| Check | Pass |
|-------|------|
| Invariants hold | Constraints consistent |
| Computations correct | Algorithms accurate |
| Edge cases handled | Boundaries and empty cases correct |

## Process

1. Identify invariants and trace logic flow
2. Check edge cases and fix violations
3. Run tests if they exist

## Output

Report: `file — issue — fix`

Do not ask for confirmation. Analyze and implement directly.
